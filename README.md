# 游戏游玩教学：
  1.将resources压缩包解压至项目目录（2.0无需）  
  2.运行test.py  （2.0运行main.py即可）
  3.（2.0更新）F4查看控制说明。注：游玩需更改win输入法改为美式键盘（非gbk即可）如出现无法操作，请按enter（主键盘）
